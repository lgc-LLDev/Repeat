# Repeat

+1 复读机

可能会与一些聊天插件冲突，`+1` 消息可能还是会被显示，请酌情使用

## 效果图

![1](https://raw.githubusercontent.com/lgc-LLDev/readme/main/Repeat/1673947551735.gif)

## 安装

```bash
lip install github.com/lgc-LLDev/Repeat
```

## 配置

实际的配置文件请不要有注释

### `plugins/Repeat/config.json`

```jsonc
{
  // 触发 +1 功能的关键词
  "keywords": ["+1"],
}
```
